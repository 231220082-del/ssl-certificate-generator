# SSL Certificate Generator

## Deskripsi
Aplikasi web sederhana menggunakan PHP dan OpenSSL untuk menghasilkan sertifikat SSL (Self-Signed Certificate).

## Fitur

- Input Country
- Input State / Province
- Input Locality / City
- Input Organization Name
- Input Common Name
- Generate RSA Private Key
- Generate Certificate (.CRT)
- Masa berlaku sertifikat 365 hari

## Teknologi

- PHP
- OpenSSL
- HTML
- CSS

## Cara Menjalankan

1. Simpan project di folder `htdocs`.
2. Jalankan Apache pada XAMPP.
3. Buka:

```
http://localhost/ssl-generator/
```

4. Isi data identitas.
5. Klik **Generate Certificate**.

## Output

- Private Key
- Certificate (.CRT)
